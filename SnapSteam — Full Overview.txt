## SnapSteam — Full Overview

---

### What it is

SnapSteam is a **Windows desktop app** that lets users add Steam games to their Steam library without owning them. It does this by downloading the necessary game data files that Steam needs to recognize and launch a game, then placing them in the right folder automatically. The user opens Steam, and the game appears in their library as if it was installed.

---

### Who uses it

People who use Steam but want to access games through unofficial means. They authenticate using their **Discord account** — no separate sign-up or email required.

---

### How the subscription works

Access is gated by **Discord roles** on a private server. When someone subscribes via **Patreon**, they get a role assigned on Discord. The app reads that role to determine what tier the user is on. There are 5 tiers:

- **Free** — Very limited (1 download every 3 days, max 10/month)
- **Starter** — $3/month — 2/day, 20/month
- **Pro** — $6/month — 4/day, 40/month, manual game updates
- **Premium** — $10/month — 5/day, 65/month, more Online Fixes
- **Ultimate** — $15/month — Unlimited everything, automatic game updates

Each tier also controls access to extra features, not just download counts.

---

### What the app actually does (features)

- **Download games** — Search for any game by name or ID and add it to your Steam library in one click
- **Library view** — See all the games you've added through SnapSteam
- **Auto game updates** — On launch, the app silently checks if any of your games have newer data available and updates them in the background (Ultimate only)
- **Manual game updates** — A button per game to refresh its data on demand (Pro+)
- **Online Fixes** — A separate tool that fixes a specific problem where some games show "No Internet" when launched; it downloads a fresh copy of the game's data to resolve it
- **Discord DM notifications** — After a scan or update, the app sends you a direct message on Discord listing which games were updated or have updates waiting
- **Referral system** — Users get a referral code; when someone uses it, it's tracked
- **Settings** — Configure where Steam files are stored, toggle auto-updates, change language
- **Multi-language** — English, Arabic, Spanish, Urdu, Chinese
- **Changelog popup** — When a user updates to a new version, a "What's New" popup appears showing release notes

---

### The backend (invisible to users)

Everything runs on **Cloudflare's serverless infrastructure** — there are no traditional servers to maintain. When the app does anything (login, download, update check), it talks to this backend which:

- Verifies the user's Discord identity and subscription tier
- Enforces daily and monthly download limits
- Connects to an external service that actually generates/serves the game data files
- Tracks usage per device
- Fires Discord alerts to the owner when something suspicious happens (e.g. a user hitting rate limits aggressively)
- Stores all configuration in a database

---

### The admin dashboard

A private web page (not in the app) that the owner uses to manage everything:

- **Version control** — Set the current app version, the minimum supported version (forcing old users to update), and write changelog notes with a live preview
- **User management** — Look up any user, see their usage, ban or unban them
- **Referrals** — See all referral activity
- **Branding** — Change displayed app name and logo URL
- **Pricing** — Edit what features each tier shows in the pricing modal
- **Limits** — Adjust download limits per tier without redeploying
- **Links** — Update external links (Discord server, Patreon, etc.)
- **Discord webhook** — Set where security alerts get sent
- **Environment config** — Manage backend settings
- **Raw JSON** — Direct config editing for advanced use

---

### The full picture

SnapSteam is essentially a **subscription SaaS product disguised as a desktop utility**. The app is the product users interact with. Discord is the identity and membership layer. Patreon is the payment processor. Cloudflare is the backend. The admin dashboard is the control panel. All of it works together without a traditional server, a login system, or a payments team — it's built lean so one person can run it.
